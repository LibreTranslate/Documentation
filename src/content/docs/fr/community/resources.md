## Forum

Toutes les discussions liées au projet ont lieu sur le forum communautaire. C'est aussi le meilleur endroit pour demander de l'aide. Venez dire bonjour !

https://community.libretranslate.com

## GitHub

Le code source du projet est hébergé sur GitHub. Le système de suivi des problèmes peut être utilisé pour signaler des problèmes, mais pas pour poser des questions. Utilisez plutôt le forum.

https://github.com/LibreTranslate/LibreTranslate

## BlueSky

Nous avons un compte BlueSky à faible volume que nous utilisons pour diffuser les dernières nouvelles du projet.

https://bsky.app/profile/libretranslate.com

## Weblate

Vous pouvez aider à traduire l'interface utilisateur de LibreTranslate via Weblate.

https://hosted.weblate.org/projects/libretranslate/app/
