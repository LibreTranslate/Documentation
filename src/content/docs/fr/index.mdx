LibreTranslate est une API de traduction automatique libre et open source. Contrairement à d'autres API, elle ne dépend pas de logiciels propriétaires tels que Google ou Azure pour effectuer des traductions. Au lieu de cela, son moteur de traduction est propulsé par la librairie open source [Argos Translate](https://github.com/argosopentech/argos-translate).

:::tip[Essayez-le en ligne !]
Ouvrez [libretranslate.com](https://libretranslate.com) dans votre navigateur.
:::

## Démarrage rapide

### Auto-hébergé

Si vous souhaitez exécuter LibreTranslate gratuitement sur votre ordinateur :

```bash
$ pip install libretranslate
$ libretranslate
$ curl -X POST http://localhost:5000/translate -d q="Hello" -d source=en -d target=es
```

```json
{"translatedText":"Hola"}
```

Voir aussi la page d'[installation](/fr/guides/installation/).

### Hébergement géré

Si vous souhaitez une installation plus facile, une haute disponibilité, des traductions rapides et un service géré professionnellement pour un petit coût mensuel :
 * Obtenez une clé API sur : https://portal.libretranslate.com

```bash
$ curl -X POST https://libretranslate.com/translate -d q="Hello" -d source=en -d target=es -d api_key=xxxxx
```

```json
{"translatedText":"Hola"}
```

Dans tous les cas, nous sommes heureux que vous soyez ici ! ❤
